# 三模型全流程部署教学文档

目标：从零开始在本机完整部署 Emotion-LLaMA、AffectGPT、Emotion-Qwen，并验证能跑通推理入口。文档按“环境 → 权重 → 配置 → 运行 → 验证 → 常见问题”组织，适合教学与复现。

## 0. 前置准备

### 0.1 目录结构与路径
- 工作根目录：`/path/to/Modals2602`
- 三模型目录：
  - `/path/to/Modals2602/Emotion-LLaMA`
  - `/path/to/Modals2602/AffectGPT`
  - `/path/to/Modals2602/Emotion-Qwen`

### 0.2 磁盘与 GPU 基线
- 预留空间：至少 200GB（三模型权重与缓存较大）
- GPU：建议 ≥ 24GB 显存，若不足可调低分辨率或帧率

### 0.3 账户权限与 Token
- Emotion-LLaMA 依赖 Llama-2 权重，需要 Hugging Face 账号授权并配置 Token

示例（仅示意，不包含真实 Token）：
```bash
export HF_TOKEN=YOUR_HF_TOKEN
```

## 5. 常见问题排查

### 5.1 权重下载失败
- Hugging Face 权限不足会导致 401/403，需确保有 Llama-2 访问权限并设置 HF_TOKEN

### 5.2 CUDA OOM
- 降低视频分辨率或帧率
- Emotion-Qwen 可设置 `max_pixels` 限制输入尺寸

### 5.3 路径问题
- 所有权重路径必须与本机实际位置一致
- Windows/WSL 路径不兼容时需改为 Linux 绝对路径
